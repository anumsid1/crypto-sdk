# Crypto SDK

Python SDK for fetching and analyzing cryptocurrency market data.

## Features

- Fetch market data from CoinGecko
- Analytics utilities
- Retry & timeout handling
- Caching support
- Async client
- Unit tests

## Installation

pip install -e .

## Usage

python main.py

## Validation
- `pip install pytest`
- `python -m pytest`
- `rm -rf venv`
- `python3 -m venv venv`
- `source venv/bin/activate`
 - `pip install -e .`
- `pip install build`
- `python -m build`
